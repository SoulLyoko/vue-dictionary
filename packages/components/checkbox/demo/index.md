## Props

支持 el-checkbox 所有参数和事件

- [element-ui#checkbox](https://element.eleme.cn/#/zh-CN/component/checkbox#checkbox-attributes)

| 参数    | 说明                                      | 类型                    | 可选值 | 默认值 |
| ------- | ----------------------------------------- | ----------------------- | ------ | ------ |
| v-model | 绑定值                                    | string / number / array |
| data    | 字典数据，可以是数组或返回 Promise 的函数 | array                   | -      | -      |
